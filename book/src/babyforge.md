# Baby Forge

### Description

This will describe how to use and operate Baby Forge. Baby Forge is the ability to spin up new DPU environments prior to a Forge install.

Baby Forge accomplishes the following:

1. DPU BMC FW updater
2. DPU BFB with SFC Install over a BMC (via the DPU DevOps Kit)
3. Deploy DOCA HBN Container (via the DPU DevOps Kit)
4. HBN Deployment using NVUE
5. Install the DPU Ops Tools specifically for Forge

The five steps above are written in Ansible and Python
